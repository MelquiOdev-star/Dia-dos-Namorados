# Site Romântico para Ana Julia

Este é um site simples e romântico criado especialmente para Ana Julia, com frases fofas e declarações de amor.

## Como usar

1. Abra o arquivo `index.html` em qualquer navegador web, incluindo dispositivos móveis.
2. Desfrute das frases românticas e do design feito com amor.
3. Clique nos botões para ver animações românticas!

## Estrutura do projeto

- `index.html`: Página completa com HTML, CSS inline e JavaScript inline para facilitar o compartilhamento e funcionamento em dispositivos móveis.
- `style.css`: Arquivo CSS separado (opcional, já integrado no HTML).
- `script.js`: Arquivo JavaScript separado (opcional, já integrado no HTML).

## Funcionalidades

- **Frases Românticas**: Várias declarações de amor.
- **Botões Interativos**:
  - Enviar um Beijo: Mostra um beijo flutuante.
  - Fazer Chover Rosas: Faz rosas caírem da tela.
  - Mostrar Amor Extra: Exibe uma mensagem especial.
- **Responsivo**: Otimizado para desktop, tablet e celular.

## Personalização

- Para adicionar mais frases, edite o arquivo `index.html` na seção `<section class="frases">`.
- Para alterar os estilos, modifique o `style.css`.
- Para mudar as animações, edite o `script.js`.

## Troubleshooting

- Se as fontes não carregarem, verifique sua conexão com a internet (usa Google Fonts).
- Para visualização offline, as fontes podem não carregar; considere fontes locais se necessário.
- As animações requerem JavaScript habilitado no navegador.
- O site foi otimizado para funcionar em dispositivos móveis; se ainda houver problemas, tente um navegador diferente ou habilite JavaScript.
